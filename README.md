# Bluegg-Tabs

Simple tabs

## Usage

```shell
# install
npm install --save bluegg/bluegg-tabs
```

```js
// import - common.js
var Tabs = require ('bluegg-tabs');

// Get your Tabs element
var myTabEl = document.querySelectorAll('.tabs')

// Init the tabs
new Tabs(myTabEl);
```
